To fix the bad normals (hacky shit but it works for the most part)

Turn off Normals Auto Smooth 	(Blender)
Export the model as FBX 	(Blender)
Import the FBX file 		(TagTool)
Extract the model 		(TagTool)
Import the AMF model 		(Blender)
Turn on Normals Auto Smooth 	(Blender)
Export the model as FBX 	(Blender)
Import the FBX file 		(TagTool)
Profit!!
